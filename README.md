# Teslo API

1. Clonar proyecto
```
npm install
```

2. Clonar el archivo ```.env.template``` y renombrarlo a ```.env```

3. Cambiar variables de entorno

4. Levantar la base de datos
```
docker-compose up -d
```

5. Ejecutar SEED
```
http://localhost:3000/api/seed
```

6. Levantar el proyecto
```
npm run start:dev
```
